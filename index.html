Чур меня!

лесной перескрёсток

для первых свиданий

печка у избы

для уютного вечера

лесной переулок

для первых прогулок

берег реки

для русалок
и водяных

мрачный холм

для драматичных натур
агенство знакомтсв
для славянских
существ
обряды свиданий

воздух

для любителей
экстрима

вода

для любителей
поплавать

огонь

для любителей
погорячее

земля

для любителей
спокойствия

сказка — ложь, да в ней намёк сказка — ложь, да в ней намёк сказка — ложь, да в ней намёк
чур меня! чур меня! чур меня! чур меня! чур меня! чур меня!
